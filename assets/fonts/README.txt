Fonte Variável Hanken Grotesk
============================

Este download contém Hanken Grotesk como fontes variáveis ​​e fontes estáticas.

Hanken Grotesk é uma fonte variável com este eixo:
  peso

Isso significa que todos os estilos estão contidos nesses arquivos:
  HankenGrotesk-VariableFont_wght.ttf

Se seu aplicativo oferece suporte total a fontes variáveis, agora você pode escolher estilos intermediários
que não estão disponíveis como fontes estáticas. Nem todos os aplicativos oferecem suporte a fontes variáveis ​​e
nesses casos, você pode usar os arquivos de fonte estáticos para Hanken Grotesk:
  static/HankenGrotesk-Medium.ttf
  static/HankenGrotesk-Bold.ttf
  static/HankenGrotesk-ExtraBold.ttf

iniciar
-----------

1. Instale os arquivos de fonte que deseja usar

2. Use o seletor de fontes do seu aplicativo para visualizar a família de fontes e todas as
estilos disponíveis
